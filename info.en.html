Contacting the Developers

If you want to contribute to the NetSurf project, please see the Developer Information. Before submitting a bug report, please check that you have the latest version and check to see if the bug has already been discovered. Bug reports and questions about NetSurf can be raised in the following forms.

Mailing List

The netsurf-develop mailing list is for discussion of NetSurf. If you are interested in NetSurf's development, please subscribe to this list.

IRC Channel

The #netsurf-develop IRC channel on the IRCNet network is also for discussion of NetSurf.

Bug Tracker

Please submit any bugs to the bug tracker.

Current Features & Limitations

The following features have been implemented:

  • Basic HTML 4, including tables
  • Parts of CSS
  • External CSS files
  • <style> element
  • Local and HTTP fetching
  • Encrypted HTTP or https
  • HTML forms, using GET and POST methods
  • Cookie support
  • Browser authentication
  • Redirects
  • Memory cache
  • Plugin System
  • View Source (Press F8)
  • Save as Drawfile (Press Ctrl+Shift+F3)
  • Support for Acorn URI Protocol and URI files

The following is in progress:

  • GUI
  • PNG Support
  • JPEG Support
  • GIF Support
  • File downloads (eg. zip files)
  • Improved Memory handling

To see what remains to be done in more detail, please see the TODO lists.

Known Bugs

  • No way to re-download a webpage, if it has been updated, without quitting NetSurf.
  • PNGs with transparency don't display correctly unless you have Select.
  • NetSurf quits when it runs out of memory
